# Playstore-data-analysis-
Data cleaning &amp; Exploratory Data Analysis of Google Play Store apps to identify patterns in ratings, installs, categories, and pricing and extracting maximum possible key insights from data .
# Google Play Store Apps — Exploratory Data Analysis

## Objectives

* Clean and preprocess real-world dataset
* Handle missing values and duplicates
* Convert and standardize data types
* Perform univariate and bivariate analysis
* Extract meaningful insights from app data



## Data Cleaning

* Removed missing values 
* Handled data types and converted data types wherever necessary 
* identifying the duplicated values within the large data under various features 
* Removed duplicated values and keep unique ones 
* Handled "Varies with device" values
* Renaming features
* Removing and replacing unnecessary features 

## Key Insights

* Free apps receive significantly more installs than paid apps
* Family and Game category dominates total installs
* Education apps show higher average ratings
* Price has weak correlation with ratings
* Most apps are concentrated between 1M–10M installs
